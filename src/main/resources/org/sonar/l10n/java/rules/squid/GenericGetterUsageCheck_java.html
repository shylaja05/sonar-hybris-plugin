This rule detects direct usage of getter Services in Controller. In Hybris Controller should not call getter methods of Facade.

Noncompliant Code Example


@Facade
public class PublicServiceGetterTestSampleFacade {
...
    public String getProductService() { // Noncompliant {{Remove or make private all the public getter methods and properties that make Services available to Controllers}}
        return "invalid"; 
    }
...
}

Compliant Solution

@Facade
private String getService() { // Compliant
        return "valid";
}
