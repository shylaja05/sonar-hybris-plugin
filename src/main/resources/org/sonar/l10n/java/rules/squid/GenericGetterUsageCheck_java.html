This rule detects direct usage of getter Services in Controller. In Hybris Controller should not call getter methods of Facade.

Noncompliant Code Example


public class PublicServiceGetterTestSampleFacade {
...
    public productService getProductService() { // Noncompliant {{Remove or make private all the public getter methods and properties in Facade that make Service available to Controllers}}
        return "invalid"; 
    }
...
}


public class PublicServiceGetterTestSampleService {
...
    public productService getProductDao() { // Noncompliant {{Remove or make private all the public getter methods and properties in Service that make Dao available to Controllers}}
        return "invalid"; 
    }
...
}

Compliant Solution

@Facade
private String getService() { // Compliant
        return "valid";
}
