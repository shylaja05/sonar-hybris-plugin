This rule detects direct usage of Html in Controller. In Hybris Controller should not define the HTML directly.

Noncompliant Code Example

@Controller
public clall HtmlUsageTestSampleController {
	String nonCompliantMethod() {
        return ""; // Noncompliant {{Refactor Controller to not use html tags directly and let view template define the tags}}
    }
}

Compliant Solution

@Controller
public class HtmlUsageTestSampleController {
...
    String compliantMethod() {
        return "index.jsp";
    }
...

}
